第六章

定积分的应用
元素法 · 几何 · 物理

1. 引入背景
▼
  • 目的：应用定积分解决几何、物理问题。
  • 核心方法：元素法（微元法）。
  • 回顾案例：曲边梯形面积的计算过程。
2. 曲边梯形面积计算的四个步骤（回顾）
▼
  • 分割：将区间[a,b]分割为n个小区间，总面积A分割为部分面积ΔA_i。
  • 近似：用f(ξ_i)Δx_i近似代替ΔA_i。
  • 求和：得到总面积近似值 A ≈ Σf(ξ_i)Δx_i。
  • 取极限：令λ→0，得到精确值 A = ∫ₐᵇ f(x)dx。
3. 元素法的核心思想与条件
▼

所求量U的特性：相关性、可加性、局部近似性。面积元素 dA = f(x)dx 是ΔA的主要线性部分。

4. 元素法的操作步骤（通用方法）
▼
  • 选变量定区间：选取积分变量x，确定变化区间[a,b]。
  • 找元素dU：在[x,x+dx]上，ΔU ≈ f(x)dx，则dU = f(x)dx。
  • 列积分求总量：U = ∫ₐᵇ f(x)dx。
1.1 直角坐标情形
▼
  • 基本模型：曲边梯形 y=f(x) (f(x)≥0), x=a, x=b 的面积 A = ∫ₐᵇ f(x)dx。
  • 面积元素：dA = f(x)dx。
  • 复杂图形处理：X-型区域 dA = [上曲线 - 下曲线] dx；Y-型区域 dA = [右曲线 - 左曲线] dy。
  • 例1：y²=x 与 y=x² 围成的面积。
  • 例2：y²=2x 与 y=x−4 围成的面积（选择y作变量更简便）。
  • 例3：椭圆 x²/a² + y²/b² = 1 的面积（利用对称性，参数方程换元得 πab）。
1.2 极坐标情形
▼
  • 基本模型：曲边扇形 ρ=ρ(θ), θ=α, θ=β 的面积。
  • 面积元素：dA = (1/2)[ρ(θ)]² dθ。
  • 面积公式：A = ∫_α^β (1/2)[ρ(θ)]² dθ。
  • 例4：阿基米德螺线 ρ=aθ 从0到2π与极轴围成的面积。
  • 例5：心形线 ρ=a(1+cosθ) 所围成的面积（利用对称性）。
2.1 旋转体的体积
▼
  • 绕X轴旋转：体积元素 dV = π[f(x)]² dx，体积 V = ∫ₐᵇ π[f(x)]² dx。
  • 例6：直角三角形绕X轴旋转成圆锥的体积。
  • 例7：椭圆绕X轴旋转成旋转椭球体的体积。
  • 绕Y轴旋转：体积公式 V = ∫_c^d π[φ(y)]² dy。
  • 例8：摆线一拱分别绕X轴和Y轴旋转的体积（绕Y轴时用“大减小”法）。
2.2 平行截面面积已知的立体的体积
▼
  • 通用方法：若垂直于X轴的截面面积为A(x)，则体积元素 dV = A(x) dx，体积 V = ∫ₐᵇ A(x) dx。
  • 例9：平面斜截圆柱体所得立体的体积（截面为直角三角形）。
  • 例10：正劈锥体的体积（截面为等腰三角形）。
3. 平面曲线的弧长
▼
  • 弧长的概念：光滑曲线弧是可求长的，其弧长是内接折线长度的极限。
  • 参数方程 x=φ(t), y=ψ(t)：弧长元素 ds = √[φ'²(t)+ψ'²(t)] dt，弧长 s = ∫_α^β √[φ'²+ψ'²] dt。
    例12：摆线一拱的长度 = 8a。
  • 直角坐标 y=f(x)：弧长元素 ds = √[1 + (dy/dx)²] dx，s = ∫ₐᵇ √[1+f'²(x)] dx。
    例11：y=(2/3)x^(3/2) 一段弧的长度。
  • 极坐标 ρ=ρ(θ)：弧长元素 ds = √[ρ²(θ) + (dρ/dθ)²] dθ，s = ∫_α^β √[ρ²+ρ'²] dθ。
    例13：阿基米德螺线一段的弧长。
1. 变力做功
▼
  • 力 F(x) 随位置变化，功元素 dW = F(x) dx，总功 W = ∫ₐᵇ F(x) dx。
  • 典型场景：弹簧拉伸、抽水做功。
2. 液体静压力
▼
  • 压强随深度线性增加：p = ρg x。
  • 压力元素：dP = ρg x · b(x) dx（b(x)为宽度）。
  • 总压力：P = ∫ₐᵇ ρg x b(x) dx。
  • 典型场景：闸门、水坝受力。
3. 万有引力
▼
  • 引力为矢量，需分量积分。
  • 引力元素：由质量微元 dm = ρ dy 产生。
  • 利用对称性简化（如 F_y = 0）。
  • 典型场景：细杆对质点的引力。
